# Bank-fraud-detection

In this end-to-end tutorial, we work on a real-world financial fraud problem and show how machine learning models can help banks identify suspicious transactions and reduce financial losses.

We walk through the complete process, from understanding raw transaction data to building strong fraud detection models..<br>
Following things will be covered in this project:.<br>

✅ How to understand a real fraud detection business problem.<br>
✅ How to perform EDA on transaction data.<br>
✅ How to handle highly imbalanced fraud datasets.<br>
✅ How to create useful features like balance differences and high-amount flags.<br>
✅ How to identify fraud patterns using transaction type and time behavior.<br>
✅ How to build and compare Logistic Regression, Random Forest, and XGBoost models.<br>
✅ How to evaluate models using ROC-AUC, PR-AUC, Precision, Recall, and F1-score.<br>
✅ How to tune decision thresholds to reduce false positives.<br>
✅ How to check model performance using a confusion matrix and fraud recall.<br>
✅ How machine learning fraud systems help reduce financial risk in production.<br>
